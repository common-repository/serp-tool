=== Serp Tool ===
Tags: Serp tool
== Description ==
Tested up to: 4.0
You can see how google see your meta tags, meta descriptions. How google see your website. 

You can call this in any post or pages with help of shortcode. 

Shortcode for access this plugin is: [serp-tool]

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `serp.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place [serp-tool] in your pages
